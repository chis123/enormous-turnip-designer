# Anormous Turnip Designer

一个基于 Handsontable 实现的 JavaScript 版本的 Excel 。

## 兼容性

* Firefox 34
* Chrome 51
* Internet Explorer 10
* Edge 20
* Safari 9
* Opera 26

* [ ] 删除工作表
* [ ] ~~隐藏工作表~~

* [x] 插入列
